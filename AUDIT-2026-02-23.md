## Summary of Critical Issues (Must Fix)

1. **AUTH-02: Register page missing** ‚Äî Login page links to non-existent `/register`. New users can't sign up.
2. **TASK-01: Attachment file serving broken** ‚Äî `/api/files/attachments/...` returns 500. Uploaded files can't be viewed.
3. **GLOBAL-01: SSE connection errors** ‚Äî Real-time event stream constantly failing with errors every ~15-30 seconds.
4. **NAV-01: Sidebar navigation broken** ‚Äî Clicking sidebar nav items doesn't navigate to the correct workspace-scoped URL.

## Summary of Major Issues (Should Fix)

1. **LIST-01/02, BOARD-02: Generic names in heading/breadcrumb** ‚Äî "List", "Space", "Workspace" shown instead of actual names.
2. **GANTT-01: Gantt view placeholder** ‚Äî "Coming soon" text only.
3. **SET-01: Member management placeholder** ‚Äî Can't invite/manage team members.
4. **NAV-02: Space/folder/list navigation** ‚Äî Sidebar items toggle instead of navigate.
5. **WORK-01: Workload data mismatch** ‚Äî Shows 0 data despite tasks being assigned.
6. **GLOBAL-02: React hydration mismatch** ‚Äî Radix UI ID inconsistency.
7. **GLOBAL-03: API crashes with invalid IDs** ‚Äî No input validation on API endpoints.
8. **DASH-01: Overdue tasks not clickable** ‚Äî Dashboard overdue task items not navigable.

## Recommendations

### Immediate (Pre-Launch)
1. Create a `/register` page or remove the "Sign up" link
2. Fix file serving for attachments (check MinIO/S3 configuration)
3. Fix SSE endpoint reliability or add graceful degradation
4. Fix sidebar navigation to use proper workspace-scoped URLs
5. Replace generic "List"/"Space"/"Workspace" headings with actual entity names from API data

### Short-Term
1. Add input validation on all API endpoints (UUID format check)
2. Fix React hydration mismatch for Radix UI components
3. Implement Gantt view or remove the tab
4. Add member management functionality
5. Fix Workload page to show correct data
6. Make dashboard overdue tasks clickable/navigable
7. Add `DialogTitle` to search CommandDialog for accessibility

### Polish
1. Switch to production mode (remove dev overlay)
2. Add loading skeletons and error feedback
3. Clean up demo data (remove "New" and "Hi j" test tasks)
4. Seed demo with richer sample data (docs, time entries, comments)
5. Add active state highlighting to sidebar navigation
